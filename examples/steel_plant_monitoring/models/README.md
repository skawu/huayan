# 钢铁厂 3D 模型说明

本目录用于存放钢铁厂监控平台的 3D 模型文件。

## 模型文件格式

推荐使用 GLTF 格式的 3D 模型文件（.gltf 或 .glb），这是 Qt 3D 原生支持的格式。

## 模型文件列表

| 模型名称 | 文件路径 | 说明 |
|---------|---------|------|
| 高炉 | blast_furnace.gltf | 钢铁厂高炉设备模型 |
| 转炉 | converter.gltf | 钢铁厂转炉设备模型 |
| 轧钢机 | rolling_mill.gltf | 钢铁厂轧钢机设备模型 |

## 模型要求

1. 模型文件大小建议不超过 10MB，以保证加载性能
2. 模型多边形数量建议控制在 10,000 以下，以保证渲染性能
3. 模型应包含基本的材质和纹理，以提高可视化效果
4. 模型坐标系应符合右手坐标系，Y 轴向上

## 替换说明

用户可以根据实际需求，替换为自己的 3D 模型文件，只需保持文件名一致即可。
