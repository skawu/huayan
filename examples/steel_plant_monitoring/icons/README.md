# 钢铁厂工业级图标说明

本目录用于存放钢铁厂监控平台的工业级图标文件。

## 图标文件格式

推荐使用 SVG 格式的图标文件，以保证在不同分辨率下的显示效果。

## 图标文件列表

| 图标名称 | 文件路径 | 说明 |
|---------|---------|------|
| 高炉图标 | blast_furnace.svg | 高炉设备状态图标 |
| 转炉图标 | converter.svg | 转炉设备状态图标 |
| 轧钢机图标 | rolling_mill.svg | 轧钢机设备状态图标 |
| 温度图标 | temperature.svg | 温度参数图标 |
| 压力图标 | pressure.svg | 压力参数图标 |
| 流量图标 | flow.svg | 流量参数图标 |
| 转速图标 | speed.svg | 转速参数图标 |
| 告警图标 | alarm.svg | 告警状态图标 |
| 运行图标 | running.svg | 设备运行状态图标 |
| 故障图标 | fault.svg | 设备故障状态图标 |
| 待机图标 | standby.svg | 设备待机状态图标 |

## 图标要求

1. 图标应采用工业级设计风格，简洁明了
2. 图标颜色应符合工业监控系统的视觉规范
3. 图标应包含正常、警告、错误三种状态的变体
4. 图标大小建议为 24x24 像素，以适应界面显示

## 替换说明

用户可以根据实际需求，替换为自己的图标文件，只需保持文件名一致即可。
